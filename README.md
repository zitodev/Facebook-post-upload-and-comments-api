**POST UPLOAD AND COMMENT API
**
A RESTful API for uploading posts, managing comments and handling user interaction. Built with Nodejs, Express, and MongoDB.
  Features
. User authentication and authorization 
. Create, updates, and delete posts
. Upload images/file for posts
. Add comments to posts
. Fetch all post and comments
. Protected routes using JWT
. Mongodb database integration 

TECH STACK
. Nodejs
. Express
. MongoDB
. Mongoose
. JWT Authentication
. Multer(upload)
. dotenv
. bcryptjs

**API Endpoints
****Auth Routes
**
Register User
. Post /api/auth/register;

Login User
. Post /api/auth/login;

POST Routes
Create post
.Post /api/posts/create-post;

Get all post
. Get /api/posts/posts;

Add comments
Post /api/posts/create-comment/:postId';
